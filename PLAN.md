# BioApex · 生命登顶 开发方案

> Apex 家族第三款(物理→化学→生物)。参考 ChemApex/PhysicsApex 架构与设计语言,
> 但按生物学科特性"换心脏"。bundle id `com.bioapex.app`。

## 一、产品内核:不做题海,做"考点全覆盖 + 掌握闭环"

App 是一张覆盖初高中全部生物考点的**掌握地图**。每个考点走闭环:
**学(看懂)→ 验(1–3 题精准诊断)→ 记(SM-2 间隔复习)→ 标记掌握**。
题只用于验证与触发复习。承诺:**覆盖率拉满 = 拿下高考全部分数点**。

- 主战场:高考生物(高中,~80%);初中做低门槛入口;选必3/竞赛做拔高钩子。
- 一句话:从一颗细胞到一片生态,从孟德尔的豌豆到你的基因——**看见生命如何运转**。

## 二、生物的灵魂(为何不能照抄物化)

| 灵魂 | 模块 |
|---|---|
| 看见过程(光合/呼吸/分裂/转录翻译都是"动"的) | **过程剧场** 🌟 |
| 机制与调节(稳态、负反馈、系统思维) | **稳态回路模拟器** |
| 遗传推理与概率(唯一强逻辑+强计算) | **遗传神探 + 遗传秒算** |
| 科学探究(对照/变量/预测) | **探究实验台** |
| 海量且易混的记忆 | **易混辨析 + 智能复习** |
| 跨九个数量级的生命 | **生命的尺度(缩放)** |
